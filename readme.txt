--------Installation & Run---------

**** windows ****
1. Install Firefox
2. Click main.exe


**** Mac Os ****
1. Install Firefox
2. Install python
3. Open Terminal > move to script directory > input command "pip install -r requirements.txt" or "pip3 install -r requirements.txt"
4. input command "py main.py" or "python3 main.py"

--------How to use bot-------

1. Input instagram's username & password (click save button for remember user)
2. Input keyword for search and input filter values(max followers, min followers, max likes, min likes)
3. Click Start button
4. When the search is finished or the Stop button is pressed, the results are stored in the output folder.
